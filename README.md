### HC智慧家园小程序

hc智慧家园小程序 是 HC小区开源项目的一个分支项目，主要给业主使用，其中包括缴物业费，缴停车费，投诉建议，家庭成员维护，房屋保修
房屋出租，公告查看，小区圈等功能。
